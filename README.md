# CHART-LAYOUT-BUILDER 

  "CHART-LAYOUT-BUILDER" 는 다양한 차트를 사용자화하는 기능을 제공합니다."<br>
 사용자화 한 차트는 JSON 형태로 추출되어 "BI-REQUIRED" 에서 대시보드를 구성하는데 사용됩니다.<br>
 퀵가이드를 보면서 빠르게 대시보드를 구성하고, 사용자화하여 "BI-REQUIRED" 에서 사용해보세요!
